一个完整的JavaScript实现由下列三个完成：

  • 核心(ECMAScript)
  • 文档对象模型(DOM)
  • 浏览器对象模型(BOM)

1.基本概念

1.1数据类型

ECMAScript有五种简单的数据类型（也称为基本数据类型）：Undefined、Null、Boolean、Number和String。其中还有一种复杂的数据类型-Object，Object本质上是由一组无序的明值对组成。

typeof操作

对一个值使用typeof操作符可能返回下列某个字符串：

  • "undefined":如果这个值未定义；
  • "boolean":如果这个值是布尔值；
  • "string":如果这个值是字符串；
  • "number":如果这个值是数值；
  • "object":如果这个值是对象或Null；
  • "function":如果这个值是函数；

1.2语句

1.2.1 for-in语句

for-in语句是一种精准迭代语句，可以用来枚举对象的属性。以下是用法：


  for(var propName in window){
      document.write(propName);
  }

用for-in循环显示BOM中window对象的所有属性。每次执行循环时，都会想window对象存在的一个属性名赋予变量propName,直到所有属性被枚举一遍为止。与for语句类似，控制语句中的var操作符也不是必须的。

1.2.2 label语句

该语句在代码中添加标签，以便将来使用。以下是用法：


  var num = 0;
  outermost:
  for (var i=0; i<10; i++){
      for (var j=0; j<10; j++){
          if (i == 5 && j == 5){
              break outermost; //continue outermost;
        }
          num++;
    }
}
alert(num);                 //55 or 95

label由break或continue引用。

1.2.3 with语句

将代码的作用域设置到一个特定的对象中。以下是用法：


  var a = location.search.substring{1};
  var hostName = location.hostname;
  var url = location.href;

上面代码都包含location对象，如果使用with语句，则可以改写代码如下：


  with(location){
      var a = search.substring{1};
      var hostName = hostname;
      var url = href;
    }

2.引用类型

2.1 Object类型

创建Object实例的方式有两种，一种是使用new操作符后跟Object构造函数。另一种是使用字面量，该语法应用更广泛，因为语法要求的代码量少，而且能给人封装数据的感觉。是向函数传递大量可选参数的首选方式。


  var person = {
      "name" : "Young",//等价于name : "Young",
      "age"  : 22,
      5 : true
};
//或者（不推荐）：
  var person = {};
      person.name = "Young";
      person.age = 22;

使用对象字面量语法时，属性名也可以使用字符串。访问属性对象时一般使用点表示法，也可以是方括号表示法（不推荐）。

2.2 Array类型

转换方法：toString( );

2.2.1 栈方法

push( )方法可以接收任意数量的参数，把它们逐个添加到数组末尾，并返回修改后的数组长度。而pop( )方法则是从数组末尾移除最后一项，减少数组的length长度，然后返回移除的项。

2.2.2 列队方法

shift( )方法移除数组中的第一项并返回该项，同时数组长度减一。而unshift( )方法则是在数组前端添加任意个项并返回新数组的长度。

2.2.2 重排序方法

  • reverse( ): 反转数组项的顺序。
  • sort( ): 默认情况下，该方法按升序排列数组的项-即最小的项位于最前面，最大的值排在最后面。该方法调用toStrig()得到字符串并进行比较排序。但该方法不是最佳，用一个比较函数：

  function compare (val1,val2){
      if (val1 < val2){
          return -1;
    }
      else if (val1 > val2){
          return 1;
    }
      else{
          return 0;
    }
}

  var value = {0, 1, 5, 10, 15};
  value.sort(compare);
  alter(value); //0,1,5,10,15

2.2.4 操作方法

  • contat( ): 先创建当前数组的一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。
  • slice( ): 接受一个或两个参数，即返回项的起始和结束位置。只有一个参数情况下，返回指定位置到当前数组的末尾所有项。若有两个参数，返回起始到结束之间但不包括结束位置的所有项。注意，slice( )不会影响原始数组。
  • splice( ): 最强大的数组方法。用途是向数组中部插入项，方式有如下三种：
    1. 删除： 可以删除任意数量的项，只需指定两个参数：要删除的第一项的位置和删除的项数。例如：splice(0,2)会删除数组中的前两项。
    2. 插入：指定位置插入任意数量的项，只需提供三个参数：起始位置、n（要删除的项数，为0则不改变原数组内容插入）和要插入的项。例如：splice(2, 0, "red", "blue")会从当前数组的位置2开始插入字符串"red"和"blue" 。
    3. 替换： 要删除的项数与要插入的项数一样，则插入变为替换。

2.2.5 位置方法

两种方法都接受两个参数：要查找的项和（可选）表示查找起点位置的索引。两种方法都返回要查找的项在数组中的位置，没有找到的情况下返回-1。

  • indexof( ): 从数组的开头位置开始向后查找。
  • lastIndexof( )： 数组的末尾开始向前查找。

2.2.7 归并方法

reduce( )和reduceRight( )。两个方法都接受两个参数：一个在每一项上调用的函数和（可选的）作为并归基础的初始值。使用如下：


  var value = {1,2,3,4,5};
  var sum = value.reduce(function(prev, cur, index, array){
      return prev + cur;
});
alert(sum);  //15

第一次执行回调函数，prev是1, cur是2。第二次，prev是3（1加2的结果），cur是3，数组的第三项。这个过程会持续到把数组的每一项都访问一遍，最后返回结果。reduceRight( )的作用类似，只是方向相反。

2.3 正则表达式RegExp

正则表达式的匹配模式支持一下三个标志：

  • g: 表示全局（global）模式，即模式将被应用于所有字符串，而非在第一个匹配项时立即停止；
  • i: 表示不区分大小写（case-insensitive）模式，记载确定匹配项时忽略模式与字符串的大小写；
  • m: 表示多行（multiline）模式，即在达到一行文本末尾时还会继续查找下一行中是否存在于模式匹配的项。

2.3.1 RegExp实例方法

  • exec( ): 接受一个参数，即要应用模式的字符串，然后返回包含第一个匹配项信息的数组；或没有匹配项的情况下返回null。返回的数组虽然是Array的实例，但包含两个额外的属性：index和input。index表示匹配项在字符串中的位置，而input表示应用正则表达式的字符串。在数组中第一项是与整个模式匹配的字符串，其他项是与模式中的捕获组匹配的字符串。
  • test( ): 接受一个字符串参数，在模式与该参数匹配的情况下返回true；否则，返回false。

2.3.2 RegExp 构造函数属性

2.4 String类型

2.4.1 字符串方法

用于访问字符串中特定字符的方法是charAt( )和charCodeAt( )。

2.4.2 字符串位置方法

两种方法：indexOf( )和lastIndexOf( )。第一个从开头向后搜索字符串，第二个从字符串的末尾向前搜索子字符串。

2.4.3 trim( )方法

创建一个字符的子副本，删除前缀和后缀的所有空格，然后返回结果。

2.4.4 localeCompare( )

比较两个字符串并返回下列值中的一个：

  • 如果字符串在字母表中应该排在字符串参数之前，怎返回一个负数，大多数情况是-1。
  • 如果字符串等于字符串参数，怎返回0。
  • 如果字符串在字母表中应该排在字符串参数之后，则返回一个正数，大多数情况为1。
